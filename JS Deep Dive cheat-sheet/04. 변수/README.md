`10+20` 표현식을 계산(평가, evaluation)하려면 먼저 표현식을 해석(parsing)하여 리터럴과 연산자를 파악한다.
해석하면 + 연산을 수행하기 위해서 피연산자(operand)를 기억한다.
피연산자는 메모리 임의의 위치에 저장된다. 연산 결과도 저장된다.
